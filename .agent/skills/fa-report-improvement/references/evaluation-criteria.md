# FA Report Evaluation Criteria

[Complete evaluation framework with 6 dimensions]

## Grading Scale
- A級 (90-100): 優秀,可作為範本
- B級 (80-89): 良好,具備完整性
- C級 (70-79): 及格,需要改進
- D級 (60-69): 不及格,嚴重缺陷
- F級 (<60): 不合格,需重做

## Six Evaluation Dimensions

### 1. 基本資訊完整性 (15%)
- FA 編號, 工程師, 批號, 客戶資訊
- 聯絡方式, 日期, 失效率統計

### 2. 問題描述與定義 (15%)
- 問題現象, 失效條件, 影響範圍
- 嚴重性評估, 客戶影響說明

### 3. 分析方法與流程 (20%)
- 分析工具 (SEM, FIB, EDS)
- 8D 流程, 實驗設計, 對照組

### 4. 數據與證據支持 (20%)
- 測量數據, 圖表品質, 可追溯性
- 圖片標註, 數據統計分析

### 5. 根因分析 (20%)
- 根因識別, 驗證實驗, 統計方法
- 多重驗證證據, 根因確認依據

### 6. 改善對策 (10%)
- 短期對策, 長期對策, 預防措施
- 效果驗證計畫, 標準化流程
