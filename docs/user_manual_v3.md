## 🚀 標準操作流程

### 第一步：選取技能
在頁面頂端的 **「選擇優化技能」** 下拉選單中，選取您要執行的任務。目前已預載 `fa-report-improvement (v2.1.5)`。

### 第二步：上傳檔案
根據介面提示，將對應的檔案拖曳至上傳區：
1. **FA 報告**：上傳您的 `.ppt` 或 `.pptx` 原始報告。
2. **評核 JSON**：上傳對應的 8D 評核 JSON 檔案。

> [!TIP]
> **新功能：自動容錯**
> 現在如果您上傳的 JSON 結尾有多餘的逗號或格式小瑕疵，系統會自動幫您修正，不會再報錯。

### 第三步：AI 優化指示 (選填)
在 **「優化提示詞」** 欄位中，您可以輸入具體的修改指示。例如：
- *"請特別加強根因分析部分，讓該項分數提高到 90 分以上"*
- *"針對基本資訊缺失，幫我加入負責工程師的分機號碼"*

### 第四步：執行與下載
點擊 **「開始優化」**。
- 系統會先請 **GPT-OSS (20b)** 模型根據您的指示預處理數據。
- 自動觸發後端技能包，產出優化後的投影片。
- 完成後，點擊彈出的 **「下載優化後的報告」** 即可獲取結果。

## 🛠️ 常見問題與技術支援

### 1. 為什麼 AI 加工需要等比較久？
因為系統目前使用本地的 **gpt-oss:20b** 模型進行深度推理。根據您的硬體效能，這可能需要 **1 到 3 分鐘**。我們已將超時寬限放長至 5 分鐘，請耐心等候，切勿重新整理頁面。

### 2. 出現「ReadTimeout」錯誤怎麼辦？
這通常代表本地模型負載過重。請確保後台的 Ollama 服務正常運作，且電腦沒有同時執行高耗能任務。

### 3. 如何新增自己的技能？
您只需在 `.agent/skills/` 下建立新資料夾，並撰寫符合規範的 `SKILL.md`（包含 `inputs` 定義），系統重新整理後就會自動抓取並顯示對應的 UI。
